# tetris1.0
- C++ 기반의 Console 테트리스 게임 구현 프로젝트
<br><br>


## goal
- 콘솔 환경에서 플레이할 수 있는 간단한 테트리스 게임 제작
<br><br>


## platform
- windows 10 pro / home
<br><br>


## key
- Up : 회전
- Down : 아래로 이동
- Left : 왼쪽으로 이동
- Right : 오른쪽으로 이동
- Space : 가장 아래로 이동
