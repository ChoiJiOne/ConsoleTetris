# tetris1.0
- Console 기반의 테트리스 게임
<br><br>


## 목표
- 콘솔 환경에서 플레이할 수 있는 간단한 테트리스 게임 제작
<br><br>


## 실행 플랫폼
- Windows Platform을 타겟으로 프로젝트 진행
<br><br>


## 구현 언어
- C++를 기반으로 구현 진행
<br><br>


## 플레이 키
- Up : 회전
- Down : 아래로 이동
- Left : 왼쪽으로 이동
- Right : 오른쪽으로 이동
- Space : 가장 아래로 이동
